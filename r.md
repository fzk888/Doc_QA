# Doc_QA — 基于 RAG 的本地文档问答系统

中文 | [English](./README_EN.md)

## 项目简介
Doc_QA 是一个轻量级、可离线运行的文档问答工具。它利用大语言模型（LLM）+ 检索增强生成（RAG）技术，让你通过自然语言直接“对话”本地 PDF/TXT/Markdown 等文件，无需上传云端，保护数据隐私。

## 功能亮点
- **多格式支持**：PDF、TXT、Markdown、Docx 等常见文档一键解析  
- **向量化检索**：基于中文语义向量模型（bge-large-zh-v1.5）做高精度召回  
- **离线运行**：模型与向量库全部本地部署，无需外网  
- **流式输出**：LLM 逐字返回答案，体验更流畅  
- **上下文多轮对话**：自动携带历史，支持追问  
- **一键安装**：提供 Windows / macOS / Linux 一键启动脚本
