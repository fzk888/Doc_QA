4. 测试对象评估
===============

1.  基本功能评估

> 经测试，澳电CEM的基本功能在正确的操作下基本实现，但仍存在一些轻微Bug，将有可能导致用户使用后产生不良结果。

2.  性能评估

> 资源利用率小，系统操作够流畅、稳定。

1.  CUP使用率稳定在30%左右

2.  Memory（内存）使用率稳定在27%以下

3.  Disks I/O（磁盘读写）前期浮动大，后期平稳。最高270%。
