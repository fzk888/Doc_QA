时间复杂度的计算规则
①基本操作
时间复杂度为0(1)
②顺序结构
时间复杂度按加法进行计算
③循环结构
时间复杂度按乘法进行计算
④分支结构
时间复杂度取最大值
⑤判断一个算法的效率时，往往只需要关注操
作数量的最高次项，其它次要项和常数项可以
忽略
⑥在没有特殊说明时，我们所分析的算法的时
间复杂度都是指最坏时间复杂度
所消耗的时间从小到大：
O(1) < O(logn) < O(n) < O(nlogn) < O(n2) <
O(n3)
