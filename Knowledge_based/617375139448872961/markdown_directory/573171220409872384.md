**引言**
========

在互联网时代下,我们始终坚持创新,"推陈出新,革故鼎新"才是企业不断成长进步的动力来源.星云文档的推出,让我们日常的文件处理从线下转变为线上,该产品特有的独立部署也将使得企业有自己的一套文件管理体系,从而使得线上文件的使用更加安全高效.

2.  **产品概述**
    ============

    1.  **背景**
        --------

随着当前的国际形势,制定信息安全的战略方针显得尤为重要,产品国有化的趋势越来越明显,国内许多行业都在响应国家的号召,很多产品都进行国有化改造.

信息安全,对于我们的生活也息息相关.在工作中,我们也经常使用到办公文件,对于一些安全级别较高的文件,我们往往采用人工归档的方式进行管理,这就显得不便利.而且对于一些紧急处理的文件,需要短时间进行处理也只能依靠个人进行文档编辑,在处理效率上并不高效.

**简介**
--------

「星云文档」是一款创作与分享在线文档的平台，支持文件的云存储、文件的多设备同步以及文件的多用户在线协作,是面向企业应用的产品.

它可以帮助用户通过创建文档（Word）、表格（Excel）、演示文稿(PPT)等来组织想法与表达内容，并通过云端存储和多设备同步功能来随时随地查看这些文件。其他多用户实时在线协作的拓展功能也能很好的帮助用户解决工作与生活中遇到的问题。

演示地址:
[[https://star.csii.com.cn/]{.underline}](https://star.csii.com.cn/)

**产品特点**
============

3.1文件存储
-----------

用户可存储文件在平台上,方便管理自身的文件

**产品功能**
============

5.1我的文档
-----------

针对用户自身的文档进行文件操作，同时也可发起分享与多用户协作,支持在线预览和在线编辑文件.

5.2协作办公
-----------

即多人实时在线查看和编辑一个文档。此模块为对用户操作协作办公的文件进行汇总统一处理，协作办公分为"我参与的"和"我发起的"。

5.3上传下载
-----------

用户可以从本地文件上传到云端,同时也能从云端文件下载到本地,支持断点续传和分片上传

5.4 我的分享
------------

可以分享文件给指定的用户,分享类型有公开分享和私密分享.用户分享的文件可以统一管理；

5.5我的收藏
-----------

重要文件收藏处理，收藏文件单独汇总，查找快捷，管理方便；

5.6企业团队
-----------

企业文件集中存储，统一管理,在企业成员内部有共享的文件,可参与多人在线编辑,提高办公效率

5.7标签文件
-----------

用户可以针对某一文件进行标签化,为用户提供定位文件的便捷通道

**产品优势**
============

6.1统一管理文件
---------------

当前企业规模越来越庞大,各自分区办公,对于文档的管理显得比较散乱,各自使用外部产品,没有企业自主产品来进行文件的归档管理.

6.2在线协作
-----------

普通的办公软件只能针对个人编辑,而无法面向相关人员协作,"单兵作战"远不如"团体配合"

6.3信息安全
-----------

很多办公产品都不属于国有产品,对于企业使用文件过多依赖非国有产品会有安全隐患,容易信息泄露

6.4私有部署
-----------

该产品是面向企业的产品,支持独立部署到客户企业,可定制化产品功能
