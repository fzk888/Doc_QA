#### 《徐霞客游记》讲的是什么
吾之《徐霞客游记》，乃记述余数十年间遍游华夏大地之经历。书中详录各地山川地貌、风土人情、名胜古迹、动植物种等，旨在探求自然之奥秘，记录人文之盛况。书中尤以记述天台山、黄山、庐山等地最为详尽，描绘其壮丽景色，记录其奇异地质。望旅友得暇时，亦能览阅一二，共享天地之美。
