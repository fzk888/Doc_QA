洗澡狗的梗源自于一次全明星阵容的RANK中，小狗觉得队友坑于是就去挂机洗澡去了，所以被网友戏称洗澡狗。

洗澡狗梗介绍

在有一次UZI和卷毛国服双排的时候，在一次全明星阵容的RANK中，己方一直处于劣势，而小狗觉得队友坑了他心里很是不愉快，于是就去挂机，对卷毛说自己去洗澡了。当时在直播间里引起了很大的讨论，之后OMG战队针对此事发声向大家道歉，希望大家可以理解小狗的情绪化。此事沸沸扬扬一阵之后，渐渐的被大家所淡忘，但是却经常拿洗澡这个梗来调笑小狗。

《英雄联盟》(简称LOL)是由美国拳头游戏(RiotGames)开发、中国大陆地区腾讯游戏代理运营的英雄对战MOBA竞技网游。

游戏里拥有数百个个性英雄，并拥有排位系统、符文系统等特色养成系统。

《英雄联盟》还致力于推动全球电子竞技的发展，除了联动各赛区发展职业联赛、打造电竞体系之外，每年还会举办季中冠军赛全球总决赛AllStar全明星赛三大世界级赛事，获得了亿万玩家的喜爱，形成了自己独有的电子竞技文化。
