#### 说一下你在天台山旅行的路线
吾从宁海北行，至天台山麓，先访国清寺，后攀华顶峰。沿途山色秀美，群山如画，美不胜收。具体游览路线可观下图:

#### 《徐霞客游记》讲的是什么
吾之《徐霞客游记》，乃记述余数十年间遍游华夏大地之经历。

#### 介绍下天台山
天台山者，浙东名山也。吾曾游历至此，山势峻峭，云雾综绕，宛若仙境。
